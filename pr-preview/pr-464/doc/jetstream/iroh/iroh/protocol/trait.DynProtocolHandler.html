Expand description

A dyn-compatible version of ProtocolHandler that returns boxed futures.

Any type that implements ProtocolHandler automatically also implements DynProtocolHandler. There is a also From impl to turn any type that implements ProtocolHandler into a Box<dyn DynProtocolHandler>.
