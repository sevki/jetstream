Enum MaybeTlsStream 

#[non_exhaustive]
pub enum MaybeTlsStream<S>
where
    S: Read + Write,
{
    Plain(S),
    Rustls(StreamOwned<ClientConnection, S>),
}
Expand description

A stream that might be protected with TLS.
